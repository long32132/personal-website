### 作品一：财务 AI 小助理

**分类：** AI Application / 财务效率工具

面向项目应收、应付管理设计的财务助手。

上传 Excel 后，系统会自动整理应收应付数据，计算未收未付金额、逾期天数和临近付款事项，并用看板呈现重点风险。用户也可以直接用自然语言查询项目回款、逾期情况和财务摘要。

常见查询由本地规则快速处理；需要判断和建议的复杂问题，则结合财务数据上下文调用大模型完成分析。

**功能标签：**

- Excel 数据解析
- 应收应付看板
- 逾期风险提示
- 自然语言查询
- 大模型 API
- 本地数据处理

**展示方式：** 使用真实界面截图；首版不放无效的体验链接。

### 作品二：久坐提醒

**分类：** Vibe Coding / 桌面工具

一个常驻系统托盘的久坐提醒工具。

它可以按照设定的时间提醒用户起身活动，并支持延后提醒、工作时间设置、定时暂停、开机自启和每日起身统计。

相比频繁打断工作的普通闹钟，它选择用安静常驻、右下角滑入和温暖反馈的方式完成提醒。

**功能标签：**

- 系统托盘
- 自定义间隔
- 工作时间
- 定时暂停
- 每日统计
- Windows 桌面程序

**查看源码：** https://github.com/long32132/stand-up-reminder

### 模型回答事实核验工作流

**分类：** Dify Workflow / Fact Checking

用于批量验证模型生成内容的真实性。

工作流接收批量模型回答，分别调用博查、百度和微软搜索获取外部证据，通过多来源结果交叉核验，最终输出真实性结论、判断理由、证据来源和置信度。

### 流程

批量导入 → 多来源检索 → 证据交叉核验 → 真实性判断 → 结构化输出

### 输出内容

- 真实性结论
- 判断理由
- 证据来源
- 置信度
- 批量核验结果

这个工作流把原本需要人工逐条搜索、对比和记录的工作整理成标准化流程，在提高核验效率的同时保留判断依据，方便后续复查。
