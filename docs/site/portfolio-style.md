# 张景隆个人网站视觉规范

## 方向名称

温暖编辑感（Warm Editorial）

## 视觉原则

- 像一本可以浏览和翻阅的个人作品刊物，而不是简历、企业官网或 AI 实验室。
- 以真实内容和作品卡为主角，装饰只用于建立节奏和个人气质。
- 页面温暖、克制、有故事感；不使用发光、粒子、霓虹渐变和技能进度条。

## 颜色

- 页面背景：`#F4F0E8`
- 主文字：`#1E1D1A`
- 次文字：`#67635C`
- 强调色：`#D8613C`
- 强调色浅层：`#E9C95B`
- 卡片背景：`#F8F5EE`
- 深色作品面板：`#24241F`
- 边框：`#A7A096`

## 字体

- 中文大标题：系统宋体优先，`STSong`, `Songti SC`, `Noto Serif SC`, serif。
- 中文正文：系统无衬线优先，`PingFang SC`, `Microsoft YaHei`, sans-serif。
- 英文标签：Georgia 或等宽/小型大写风格，增加刊物感。
- Hero 标题使用高对比大字号，但手机端必须控制换行，不出现单字孤行。

## 布局

- 最大内容宽度：`1480px`。
- 页面左右留白：桌面端 `clamp(24px, 4vw, 72px)`，手机端 `20px`。
- Hero：桌面端左侧大标题、右侧简短介绍与按钮；手机端改为上下结构。
- 财务 AI 小助理：横跨两列的主作品卡，文字和真实界面截图并排。
- 久坐提醒与大话骰子：两张竖向次作品卡。
- 小红书内容：独立横向卡片，使用真实双封面截图。
- 工作流：横向五步流程，手机端变为纵向步骤。
- 移动端断点：`760px`。

## 组件

### 导航

- 顶部细边框导航，左侧显示圆形“张”字标和“张景隆”。
- 右侧为“关于我、作品、工作流、联系我”。
- 当前栏目可使用陶土橙短下划线，不使用胶囊式应用导航。

### 按钮

- 主按钮为墨色实心、暖白文字、圆角控制在 `999px` 或较小圆角之一，整站统一。
- 次按钮保持透明或文字链接，使用下划线/箭头表达可点击。
- 悬浮时只做轻微位移或颜色变化。

### 作品卡

- 纸张卡片、细边框、轻微错位阴影。
- 分类标签使用陶土橙小型大写风格。
- 主作品卡允许深色产品界面区域，形成编辑版面的视觉重点。
- 所有项目说明使用真实文案，不放虚构数据。

### 工作流

- 使用编号、细线和箭头串联五个步骤。
- “证据交叉核验”步骤使用陶土橙强调，其余保持纸张色。

## 动效

- 页面进入：内容轻微上移并淡入，持续 `400–600ms`。
- 卡片悬浮：最多上移 `4px`，阴影轻微增强。
- 链接箭头：水平移动 `2–4px`。
- 必须支持 `prefers-reduced-motion: reduce`，关闭非必要动画。

## 禁止项

- AI 分身与聊天入口
- 简历时间线和技能进度条
- 霓虹色、发光边框和粒子背景
- 虚构产品截图、数据指标和客户评价
- 大面积无意义英文与商业口号
